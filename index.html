Workflow

1
Open Tool 01
Drop your .idml file. Conversion runs in the browser — nothing uploaded.
2
Download ZIP
Get index.html, styles.css, images/ folder and a MISSING-IMAGES.txt manifest.
3
Add Images
Copy your original image files into the images/ folder as listed in the manifest.
4
Open Tool 02
Drop the HTML + images, set chapter layout, click Preview then Print → Save as PDF.
