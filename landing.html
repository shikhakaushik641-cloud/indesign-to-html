Capabilities

Everything built in

No extensions, no plugins, no build step. Open index.html and start converting.

IDML converter
Full paragraph and character style mapping. Spread ordering by Y-position. Base64 image embedding. 41+ styles supported.
PDF converter
Font-size threshold heading detection. Line merging into paragraphs. Raster image extraction. Page range selection.
Figure extractor
Auto-detect vector diagram regions by text gap analysis. Manual crop drawing. Inject into vector-figure placeholders.
WYSIWYG editor
Bold, italic, underline, colour, highlight, font size, line spacing, alignment, lists, format painter.
Equation panel
Fractions, superscript, subscript, square roots, vectors, integrals, summations. Greek alphabet and 20+ operators.
Table editor
Insert tables from a grid picker. Floating toolbar to add/delete rows and columns in place.
PDF side-by-side
Load the original PDF alongside the HTML editor. Page navigation, zoom, fit-width. Drag the splitter to resize.
100% client-side
No server, no upload, no data leaves your device. Works on GitHub Pages. PDF.js + JSZip only dependencies.

Guide

How to use it

The complete workflow takes under 5 minutes for a typical chapter.

1
Open index.html in Chrome or Edge
Double-click the file or serve from GitHub Pages. No install required.
2
Select your source type
Choose IDML for full style fidelity, or PDF if you only have the exported file.
3
Upload files and convert
For IDML: upload the .idml and all linked image files from the InDesign Links folder. For PDF: set heading thresholds and click Convert.
4
Extract vector figures (optional)
Drop the original PDF to auto-detect diagram regions. Draw manual crops for anything missed. Inject into the HTML.
5
Edit and download
Fix any conversion errors in the WYSIWYG editor. Download a single self-contained HTML file — ready to distribute.
IDML path
Recommended

Output format note
The downloaded HTML embeds all images as base64 data URIs. The file is 100% self-contained — no external assets needed. It opens in any browser and can be uploaded to any web server.
